
# UnConvert - Android app
It is a Mobile App which convert the Kilogram into Pound.

Note :- The inspiration has been taken from the YouTube Tutorials.




## Installation

Step 1:

```
 Go to CODE and copy the "HTTPS"
```
Step 2:
```
Open the termial 
```
Step 3:
```
Paste $ git clone <HTTPS URL>
```
